# Security Policy

## Supported Versions

This package is supported by versions
< 0.0.6.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.6   | :white_check_mark: |
| 0.0.3   | :x:                |
|< 0.0.1  | :x:                |

## Reporting a Vulnerability

To report a vulnerability, leave an issue, or post a discussion.

We'll try to get to you immediately!
